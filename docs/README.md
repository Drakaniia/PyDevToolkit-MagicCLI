# Documentation for MagicCLI

Welcome to the MagicCLI documentation directory. Here you'll find comprehensive guides and reference materials for using and developing the MagicCLI automation toolkit.

## Available Documentation

- [Global Installation Guide](GLOBAL_INSTALLATION_GUIDE.md) - Explains how to install MagicCLI globally (similar to `npm install -g`) without cloning the repository

## About MagicCLI

MagicCLI is a powerful automation toolkit that provides:
- Git operations automation
- Project structure visualization
- Web development automation tools
- Comprehensive code quality tools
- And more!

## Getting Started

After installing MagicCLI globally using the instructions in the [Global Installation Guide](GLOBAL_INSTALLATION_GUIDE.md), simply run the `magic` command from any directory to access the main menu and begin using the various automation tools.

For usage examples and feature descriptions, refer to the main [README.md](../README.md) file in the project root.
